- Để chạy chương trình, mở command line ở vị trí hiện tại và nhập:
	java -jar StudentManagement.jar
- Folder bao gồm:
	+ cscFileFolder: chứa danh sách lớp ở dạng file csv.
	+ txtFileFolder: chứa danh sách lớp ở dạng file txt.
	+ src: file source code
	+ SoLuong.txt: số lượng sinh viên của hệ thống.
	+ README.txt: file hướng dẫn.
	+ StudentManagement.jar: file nén jar.
